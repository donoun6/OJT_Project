### 📌&nbsp;기능



| 기능       | 상세설명                                                                                                                          |
|:---------|:------------------------------------------------------------------------------------------------------------------------------|
| 제품관리     | 카테고리 등록/삭제, 제품 등록/수정/삭제 <br> 제품조회 - 조건 검색                                                                                     |
| 사용자 페이지  | 상품 조회, 장바구니 기능                                                                                                                |

| 출고 관리    | 출고 내역 조회 - 전체/상품 명/ 기간설정/묶음 처리 조회 <br>매출 조회                                                                                   |
| 입고 관리    | 입고 내역 등록, 조회- 전체/상품명/기간                                                                                                       |
| 주문 내역 관리 | 조회- 전체/기간 , 전체환불, 부분환불                                                                                                        |
